# ラジオ投稿管理アプリ

## 概要
ラジオ番組に投稿するメールのネタの保存、メールの一元管理を目的としたアプリケーション
メモの内容からLLMを用いて適したラジオコーナーの検索が可能

## 使用技術
### フロントエンド
- **Framework:** Streamlit
- **HTTP Client:** requests
- **Language:** Python

### バックエンド
- **Framework:** FastAPI
- **Database:** SQLite
- **ORM:** SQLAlchemy v2.0
- **Schema:** Pydantic v2
- **LLM:** Google Generative AI (Gemini)
- **Language:** Python

## 起動方法
1. backend以下に.env.exampleを参考に.envファイルを作成する。
2. **docker-compose up -d** でコンテナのビルドと実行を行う。

## 機能一覧

### 1. メモ管理
- メモ一覧表示
- メモ作成
- メモ削除
- メモ検索

### 2. 番組管理
- 番組一覧表示
- 番組作成
- 番組削除
- パーソナリティでの絞り込み
- 番組名での部分一致検索
- コーナー一覧表示

### 3. メール作成
- AI解析による推奨コーナー
- 手動コーナー選択
- プロフィール選択
- OS標準メーラー起動

### 4. プロフィール設定
- プロフィール一覧表示（API連携）
- プロフィール作成（API連携）
- プロフィール削除（API連携）

## AI機能（Google Gemini）

### 設定方法

1. Google AI Studioで APIキーを取得
   - https://makersuite.google.com/app/apikey

2. バックエンドの `.env` ファイルに設定
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```

## 📝 使用例

### 1. メモを作成

1. ダッシュボードの「クイックメモ」に内容を入力
2. 「追加」ボタンをクリック
3. メモがデータベースに保存される

### 2. 番組を作成

1. 「番組管理」ページを開く
2. 「新規番組を登録」を展開
3. 番組情報とコーナーを入力
4. 「登録」ボタンをクリック

### 3. AIでコーナーを推奨

1. 「メール作成」ページを開く
2. メモを選択
3. AIが自動で最適なコーナーを推奨
4. 「このコーナーに投稿」をクリック

### 4. メールを送信

1. 件名と本文を編集
2. 「メーラーで開く」をクリック
3. OS標準メーラーが起動
4. メール送信
